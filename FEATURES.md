# 주머니 핵심 기능 정리

## 기술 스택

- **Vanilla JS** (프레임워크 없음) + CSS 커스텀 속성
- **localStorage** 기반 상태 관리 (자동 저장)
- **Dify** AI 에이전트 6개 (gpt-4o-mini)

---

## 핵심 기능

### 1. 온보딩 (5단계)

1. 이름 + 캐릭터 선택
2. 목표 상품 / 금액 설정 (Dify로 실제 가격 검색)
3. 카드 CSV 업로드 → Dify가 3개월 내역 카테고리 자동 분류
4. 예산 슬라이더 조정
5. 일일 예산 확정 (월 지출 - 월 저축목표) / 30

---

### 2. 일일 예산 추적

- 오늘 지출 입력 (8개 카테고리: 식비, 카페, 교통, 쇼핑, 여가, 건강, 뷰티 등)
- **하루 정산** 버튼 → 성공/실패 판정
  - 성공: 저금통 금액 증가 + XP 획득
  - 실패: 저금통 금액 감소 + XP 차감
- **쉴드 시스템**: 30일 연속 성공 시 쉴드 1개 획득 (최대 3개), 실패 1회 방어 가능

---

### 3. 저금통 & 목표 달성

- 목표 금액 대비 현재 저축 진행률 표시
- **캐릭터 성장 5단계** (새싹 → 성장 → 단단 → 강인 → 전설)
  - 0–19% / 20–39% / 40–59% / 60–79% / 80%+
- **달력 뷰**: 초록(성공) / 빨강(초과) / 파랑(오늘), 과거 날짜 클릭 시 내역 편집 가능

---

### 4. AI 예산 분석

- Dify가 과거 카드 지출 분석 → 월 저축 목표 달성 가능한 카테고리별 예산 추천
- 카테고리별 **현재 지출 vs 권장 지출** 게이지 시각화

---

### 5. AI 트레이너 챗봇

6가지 인텐트 자동 분류:

| 인텐트 | 내용 |
|---|---|
| `SAVING_TIP` | 절약 팁 (지식베이스 검색) |
| `IMPULSE` | 충동구매 상담 + 대안 제시 |
| `APP_SERVICE` | 앱/카드 추천 |
| `GOAL_CHANGE` | 목표 변경 3단계 폼 |
| `BUDGET_CALC` | 실시간 예산 계산 + 카테고리 팁 |
| `OTHER` | 정중한 안내 + 목표 설정 유도 |

- 대화 6턴 메모리 유지

---

### 6. 캐릭터 시스템

- **8종 동물**: 강아지, 코끼리, 사막여우, 고릴라, 물범, 판다, 펭귄, 미어캣
- 레벨별 스프라이트 + 기분 상태 (어제 예산 성공 여부에 따라 good/sad)
- 레벨 3 이상 해금: 모자, 안경 코스메틱 착용 가능

---

### 7. 배지 & 미션 시스템

**22개 배지** (달성 즉시 자동 해금):

| 카테고리 | 배지 예시 |
|---|---|
| 저축 & 목표 | 첫 정산, 주간 목표, 제로데이, 저금통 5만원, 연간 약속 |
| 스트릭 | 3일 연속, 7일 출석, 30일 출석 |
| 라이프스타일 | 도시락, 배달 안 시키기, 카페 절약, 쇼핑 절제, 영수증 챌린지 |
| 커뮤니티 | 듀오, 소셜 1위, 랭킹 1위, 레벨 3/5 달성 |

---

### 8. ETF 투자 유도

- 저축 합계가 목표 금액 초과 시 **최초 1회 ETF 추천 모달** 자동 팝업 (정산 후)
- 저금통 화면에서 달성률 100% 이상이면 **ETF 배너 + 버튼** 상시 노출
- 모달 구성:
  - 정적 ETF 4종 카드 (해외: TIGER 미국S&P500·KODEX 미국나스닥100TR / 국내: KODEX 200·TIGER 코스닥150)
  - **Dify ETF 에이전트** — 초과 저축액 기반으로 지식베이스 RAG 검색 후 개인화 조언 생성
- 지식베이스: `dify_etf_knowledge.md` (국내·미국 ETF 8종 + 금액별 추천 조합)

---

### 9. 소셜 & 커뮤니티

- 친구 추가 / 수락 / 거절
- **랭킹**: 예산 성공률 기준 친구 리더보드
- **피드**: 절약 팁, 챌린지 인증 게시물
- **그룹**: 공동 목표 (아이패드 모으기, 1000만원 목표 등)

## Dify 에이전트 5개

| 에이전트 | 연결 파일 | 역할 |
|---|---|---|
| 목표설정 | `onboarding.js` | 상품 실시간 가격 검색 (Tavily) |
| 소비분석 | `onboarding.js` | 카드 내역 → 카테고리 분류 + 이상 지출 탐지 |
| 예산플래닝 | `budget.js` | 카테고리별 예산 추천 + 저축 부족분 자동 조정 |
| 트레이너 챗봇 | `trainer_chat.js` | 인텐트 분류 + 지식베이스 연동 |
| 소비요약 | `modals.js` / `mypage.js` | 하루 정산 코멘트 + 월간 리포트 |
| ETF 코치 | `modals.js` | 목표 초과 저축액 기반 ETF 추천 (RAG) |

## 상태 관리 주요 구조 (state.js)

```javascript
{
  user: { name, charName },
  goal: {
    name, targetAmount, currentSavings,
    monthlyExpenses, timelineMonths, monthlyTarget,
    categoryBudgets
  },
  dailyBudget, todaySpent, expenses[],
  piggyBalance, piggyHistory[],
  characterLevel, characterType, characterXP,
  badges[], shieldCount
}
```

주요 액션:
- `completeOnboarding()` — 목표 + 일일 예산 초기 설정
- `addExpense()` — 오늘 지출 누적
- `settleDay()` — 정산 + XP + 레벨 계산
- `updateGoal()` — 목표 변경 후 일일 예산 재계산
